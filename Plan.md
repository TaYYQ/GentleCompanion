# 应用改进计划

## 目标
实现以下功能改进：
1. 将所有小窗改为应用内页面
2. 统一所有页面为白色主题
3. 时间模式界面改为翻页时钟
4. 集成账号系统，首次打开应用时提供登录/注册/游客选项
5. 添加Apple和微软登录方式
6. 确保在使用社交或番茄钟统计时需要登录

## 技术实现细节

### 应用内导航实现
- 使用 `currentView` 状态变量管理当前显示的页面
- 使用 `switch` 语句根据不同页面类型显示相应的视图
- 实现了返回按钮功能，可从子页面返回主页面

### 白色主题实现
- 将所有背景改为白色
- 调整文本和元素颜色以适应白色背景
- 保持视觉层次和可读性

### 翻页时钟实现
- 使用 `FlipClockDigitView` 显示单个数字
- 使用 `FlipClockHalfView` 显示数字的上半部分和下半部分
- 使用 `rotation3DEffect` 实现翻页动画
- 实现了平滑的数字变化效果

### 账号系统实现
- 在激活流程中添加账号屏幕
- 提供登录、注册和游客模式选项
- 实现了完整的登录注册表单
- 添加了Apple和微软登录按钮

### 条件登录实现
- 在需要登录的功能前添加登录检查
- 实现登录提示小窗口
- 允许游客使用核心功能，仅在需要时提示登录
